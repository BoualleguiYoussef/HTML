Qu'est-ce que le CSS ?

Les feuilles de style en cascade (CSS) sont un langage pour styliser les documents HTML.

Pourquoi utiliser le CSS ?

Le CSS sépare le contenu du design, ce qui rend les sites Web plus faciles à maintenir.
